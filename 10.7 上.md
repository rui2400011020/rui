```
n=int(input())
S=[]
L=[]
answer=''
for _ in range(n):
    word=input()
    S.append(word)
    L.append(len(word))
d=min(L)
for i in range(d):
    x=S[0][i]
    flag=0
    for j in range(n):
        if S[j][i]==x:
            flag+=1
    if flag==n:
        answer+=x
    else:
        break
print(answer)
```
