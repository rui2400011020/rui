```
n=int(input())
a=[]
b=[]
for i in range(n):
   s,t=map(int,input().split())
   a.append(s)
   b.append(t)
for i in range(n):
   s=a[i]
   t=b[i]
   if s%t==0:
       print(0)
   else:
       print(t-(s%t))
```
